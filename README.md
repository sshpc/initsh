## 一个简单的Ubuntu初始化脚本


### 简单介绍

* apt升级
* 切换阿里云源（新增支持16.04 18.04 20.04）
* 简易配置静态ip
* 一键安装常用工具（screen、git、nmap）
* 输出系统信息 （新增显示磁盘信息）
* ufw简易配置 （新增持续操作）

### 快速使用
```shell
wget -O init.sh http://daili.192168.work/initsh/init.sh && chmod +x init.sh && sudo ./init.sh
```
